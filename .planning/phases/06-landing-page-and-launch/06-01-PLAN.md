**3. `supabase/migrations/012_platform_stats_rpc.sql`** — Secure RPC for anonymous counter queries:
```sql
CREATE OR REPLACE FUNCTION get_platform_stats()
RETURNS jsonb
LANGUAGE sql
SECURITY DEFINER
STABLE
AS $$
  SELECT jsonb_build_object(
    'jobs', (SELECT count(*) FROM jobs WHERE status = 'active'),
    'seekers', (SELECT count(*) FROM seeker_profiles),
    'matches', (SELECT count(*) FROM match_scores WHERE total_score >= 50)
  );
$$;

GRANT EXECUTE ON FUNCTION get_platform_stats() TO anon, authenticated;
```
This avoids RLS issues for anonymous users on seeker_profiles and match_scores tables.
