## Sampling Rate

- **After every task commit:** `pnpm test -- tests/<file>.test.ts` for the file the task touched
- **After every plan wave:** `pnpm test` (full suite — guards against regression in static-source assertions on shared files like `MarkFilledModal.tsx`)
- **Before `/gsd:verify-work`:** Full suite green + DB-level post-verify queries (read-only MCP — `pg_proc` for RPC, `pg_trigger` for cleanup trigger, `pg_indexes` for 15 FK indexes) + operator UAT 6/6 ✓
- **Max feedback latency:** ~6s for phase scope, ~95s for full suite
