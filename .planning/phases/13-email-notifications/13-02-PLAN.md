  <action>
This task requires human action — DNS record management and Resend dashboard verification cannot be done via CLI.

**Step 1: Get DNS records from Resend**
1. Log in to Resend Dashboard (https://resend.com/domains)
2. Find the sending domain (topfarms.co.nz or the configured domain)
3. If domain not yet added: click "Add Domain", enter the domain name
4. Copy the required DNS records. Resend typically provides:
   - **SPF record**: A TXT record with value like `v=spf1 include:amazonses.com ~all` (or Resend's specific include)
   - **DKIM record**: One or more TXT records with name like `resend._domainkey.topfarms.co.nz` and a long DKIM value

**Step 2: Add DNS records at domain registrar**
1. Log in to the domain registrar for topfarms.co.nz
2. Navigate to DNS management
3. Add each TXT record exactly as Resend specifies:
   - For SPF: If an SPF record already exists, MERGE the include (do not create a duplicate SPF record — domains should have only one SPF TXT record)
   - For DKIM: Add the TXT record(s) with the exact name and value from Resend
4. Save changes

**Step 3: Wait for DNS propagation**
- DNS changes can take 15 minutes to 72 hours to propagate
- Use MXToolbox (https://mxtoolbox.com/SuperTool.aspx) to check if TXT records are visible:
  - Search: `txt:topfarms.co.nz` for SPF
  - Search: `txt:resend._domainkey.topfarms.co.nz` for DKIM (adjust subdomain as shown by Resend)

**Step 4: Verify in Resend Dashboard**
1. Return to Resend Dashboard -> Domains -> topfarms.co.nz
2. Click "Verify" or check the verification status
3. Both SPF and DKIM should show "Verified" status (green checkmark)

**Step 5: Send test email**
1. From Resend Dashboard, use the "Send Test Email" feature, OR
2. Use curl to send a test email via the Resend API:
```bash
curl -X POST 'https://api.resend.com/emails' \
  -H 'Authorization: Bearer YOUR_RESEND_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "from": "TopFarms <noreply@topfarms.co.nz>",
    "to": ["your-gmail@gmail.com"],
    "subject": "TopFarms DNS Verification Test",
    "html": "<p>This is a test email to verify SPF/DKIM configuration.</p>"
  }'
```
3. Check the email arrived in inbox (not spam) in Gmail and/or Outlook
4. In Gmail: click the three dots on the email -> "Show original" -> verify SPF=PASS and DKIM=PASS
