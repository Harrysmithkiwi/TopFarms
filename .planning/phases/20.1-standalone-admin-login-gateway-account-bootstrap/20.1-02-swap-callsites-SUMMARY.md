## Decisions Made

- **Atomic single-commit refactor:** Per CLAUDE.md §4. Eliminates partial-state risk (some callsites using helper, some inline). Confirmed at commit time via `git diff --stat HEAD~1 HEAD` showing exactly 5 files.
- **Edit tool with surrounding context, not global replace:** Each `Edit` call included unique context (e.g., the `if (!session) return` line above the ternary in ProtectedRoute, the `(roleData?.role as UserRole) ?? 'seeker'` line above the ternary in VerifyEmail). Prevents Pitfall 4 (sed-style mismatch on similar-looking ternaries elsewhere).
- **Ignored Next.js plugin's `'use client'` PostToolUse warnings:** TopFarms is Vite + React Router v7, not Next.js App Router. Verified via package.json (`react-router: ^7.5.0`, no `next` dep) and `src/main.tsx` (`createBrowserRouter` from `'react-router'`). The directive would be inert.
