    Internal StarIcon helper:
    ```typescript
    function StarIcon({ filled, size = 16 }: { filled: boolean; size?: number }) {

          <path
            d="M8 1.5l1.854 3.756 4.146.602-3 2.924.708 4.128L8 10.81l-3.708 1.1.708-4.128-3-2.924 4.146-.602L8 1.5z"
            fill={filled ? 'var(--color-hay)' : 'var(--color-fog)'}
          />
        </svg>
      )
    }
    ```

    Container: `flex items-center gap-0.5`
    Each star: wrap in `<button type="button">` with minimum 44x44px tap area via padding (`p-1` on 16px icon gives ~24px; use `min-w-[44px] min-h-[44px] flex items-center justify-center` for touch target)
    Hover: use `useState<number>(0)` for hoverRating. On mouseEnter star N, set hoverRating=N; on mouseLeave container, set hoverRating=0. Display logic: `filled = i <= (hoverRating || value)`

    When `onChange` is undefined, render without button wrappers (just SVG spans) for display-only mode.

    Interface:
    ```typescript
    interface PaginationProps {
      currentPage: number
      totalPages: number
      onPageChange: (page: number) => void
      className?: string
    }
    ```

    Container: `flex items-center gap-1`

    Page button (inactive): `w-[34px] h-[34px] flex items-center justify-center rounded-[6px] border-[1.5px] border-fog bg-white text-ink font-body text-[13px] hover:border-fern transition-colors`
    Page button (active): `border-moss bg-moss text-white` (replaces border-fog bg-white text-ink)

    Prev/Next: lucide `ChevronLeft` / `ChevronRight` (16px), same 34x34px container. Disabled: `opacity-40 cursor-not-allowed`. Prev disabled when currentPage===1, Next disabled when currentPage===totalPages.
    Prev button: aria-label="Go to previous page"
    Next button: aria-label="Go to next page"

    **Sliding window algorithm:**
    - Always show page 1 and totalPages
    - Show 3 pages centered around currentPage
    - If gap between 1 and window start > 1, show "..." ellipsis
    - If gap between window end and totalPages > 1, show "..." ellipsis
    - Ellipsis: `text-[13px] text-light w-[34px] text-center` — static span, not a button

    Example: totalPages=20, currentPage=10 renders: [1] [...] [9] [10] [11] [...] [20]
