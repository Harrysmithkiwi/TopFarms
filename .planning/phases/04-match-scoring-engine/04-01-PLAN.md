**Section 1 — Add explanation column:**

ALTER TABLE public.match_scores

**Section 2 — Seeker trigger function** (`trigger_recompute_seeker_scores`):
- `RETURNS trigger`, `LANGUAGE plpgsql`, `SECURITY DEFINER`, `SET search_path = public`
- Column-level guard at top of function body: if NONE of these columns changed, `RETURN NEW` immediately:
  - `shed_types_experienced`, `region`, `sector_pref`, `visa_status`, `min_salary`, `accommodation_needed`, `couples_seeking`, `open_to_relocate`
- Use `IS NOT DISTINCT FROM` for each column comparison (handles NULLs correctly)
- For INSERT (OLD is null), skip the guard and always recompute
- Use `TG_OP = 'INSERT'` check to handle INSERT vs UPDATE
- Use CROSS JOIN LATERAL pattern to call `compute_match_score()` once per job:
```sql
INSERT INTO public.match_scores (job_id, seeker_id, total_score, breakdown, calculated_at)
SELECT
  j.id,
  NEW.id,
  (result->>'total_score')::int,
  result->'breakdown',
  now()
FROM public.jobs j
CROSS JOIN LATERAL public.compute_match_score(NEW.id, j.id) AS result
WHERE j.status = 'active'
  AND j.sector = ANY(NEW.sector_pref)
ON CONFLICT (job_id, seeker_id)
DO UPDATE SET
  total_score   = EXCLUDED.total_score,
  breakdown     = EXCLUDED.breakdown,
  calculated_at = EXCLUDED.calculated_at;
```
- Do NOT touch the `explanation` column in the upsert — it is managed separately by the AI Edge Function

**Section 3 — Seeker trigger definition:**
```sql
CREATE TRIGGER seeker_profile_match_rescore

```

**Section 6 — Backfill existing pairs:**
- Add `SET statement_timeout = '0';` before the backfill statement (prevents timeout on Supabase)
- Use same CROSS JOIN LATERAL pattern to compute and insert all active job + matching seeker pairs:
```sql
SET statement_timeout = '0';

INSERT INTO public.match_scores (job_id, seeker_id, total_score, breakdown, calculated_at)
SELECT
  j.id,
  sp.id,
  (result->>'total_score')::int,
  result->'breakdown',
  now()
FROM public.jobs j

  breakdown     = EXCLUDED.breakdown,
  calculated_at = EXCLUDED.calculated_at;

```
