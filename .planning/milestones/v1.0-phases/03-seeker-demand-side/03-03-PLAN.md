**URL-synced filter pattern:**
```typescript
const [searchParams, setSearchParams] = useSearchParams()

function handleFilterChange(key: string, value: string | string[] | null) {
  const next = new URLSearchParams(searchParams)
  if (value === null || (Array.isArray(value) && value.length === 0)) {
    next.delete(key)
  } else if (Array.isArray(value)) {
    next.delete(key)
    value.forEach(v => next.append(key, v))
  } else {
    next.set(key, value)
  }
  setSearchParams(next, { replace: true })
}
```

**Data fetching (useEffect on searchParams):**
1. Build Supabase query from params:
   ```typescript
   let query = supabase.from('jobs')
     .select('*, employer_profiles(id, farm_name, region)')
     .eq('status', 'active')
     .order('created_at', { ascending: false })
   ```
2. Apply filters from searchParams:
   - `shed_type`: `.overlaps('shed_type', shedTypes)` (GIN indexed)
   - `region`: `.in('region', regions)` if multiple, `.eq('region', region)` if single
   - `contract_type`: `.in('contract_type', types)`
   - `salary_min`: `.gte('salary_max', Number(salaryMin))` (job can pay at least seeker's min)
   - `salary_max`: `.lte('salary_min', Number(salaryMax))` (job starts below seeker's max)
   - `herd_size`: Convert bucket to range. E.g., '<200' -> herd_size_max >= 0 AND herd_size_min <= 200. Use `.or()` for multiple buckets.
   - `accommodation`: `.contains('accommodation', { available: true })`
   - `couples`: `.eq('couples_welcome', true)`
   - `visa`: `.eq('visa_sponsorship', true)`
3. Execute query. Set jobs.
4. If seeker is logged in (session + role === 'seeker'):
   - Look up seeker profile ID: `supabase.from('seeker_profiles').select('id').eq('user_id', session.user.id).single()`
   - Call batch RPC: `supabase.rpc('compute_match_scores_batch', { p_seeker_id: profileId, p_job_ids: jobs.map(j => j.id) })`
   - Build Map<jobId, MatchScore> from results.
   - Sort jobs by match score descending (best first), then by created_at descending.
5. If not logged in: skip scores, sort by created_at only.
