  <action>
    All components follow the same pattern:
    - Import `cn` from `@/lib/utils`
    - Accept `className` prop for overrides
    - Use Tailwind v4 utility classes with the theme tokens (bg-soil, text-moss, etc.)
    - Export as named export
    - Stateless, no business logic

    **Button.tsx** — 4 variants per SPEC Section 4.3:
    - Props: `variant?: 'primary' | 'outline' | 'ghost' | 'hay'`, `size?: 'sm' | 'md' | 'lg'`, extends `ButtonHTMLAttributes<HTMLButtonElement>`
    - primary: `bg-moss text-white hover:bg-fern`
    - outline: `bg-white border border-moss text-moss hover:bg-mist`
    - ghost: `border border-fog text-mid hover:border-mid`
    - hay: `bg-hay text-soil hover:opacity-90`
    - Base: `font-body text-[13px] font-bold rounded-[8px] px-4 py-2 transition-all duration-200 cursor-pointer disabled:opacity-50 disabled:cursor-not-allowed`

    **Card.tsx** — white container per SPEC:
    - Props: `className`, `children`, optionally `hover?: boolean` for hover effect
    - Styles: `bg-white border-[1.5px] border-fog rounded-[12px] p-5`
    - Hover variant: `hover:translate-y-[-1px] hover:shadow-md transition-all duration-200`

    **Tag.tsx** — 7 colour variants per SPEC Section 4.3:
    - Props: `variant: 'green' | 'hay' | 'blue' | 'grey' | 'orange' | 'purple' | 'red'`, `children`
    - green: `bg-[rgba(45,80,22,0.07)] text-moss`
    - hay: `bg-hay-lt text-[#7A5C00]`
    - blue: `bg-blue-lt text-blue`
    - grey: `bg-mist text-mid`
    - orange: `bg-orange-lt text-orange`
    - purple: `bg-purple-lt text-purple`
    - red: `bg-red-lt text-red`
    - Base: `inline-flex items-center px-2.5 py-1 rounded-full text-[11px] font-semibold font-body`

    **MatchCircle.tsx** — score display per SPEC:
    - Props: `score: number`, `size?: 'sm' | 'md' | 'lg'`
    - Sizes: sm = 38px (w-[38px] h-[38px]), md = 50px, lg = 72px
    - Colour states based on score: >=80 = moss tones (bg `rgba(45,80,22,0.08)`, border `rgba(45,80,22,0.2)`, text moss), 60-79 = orange tones, <60 = red tones
    - Number display: `font-display font-bold` with size-appropriate text
    - "MATCH" label below number (only on lg size)

    **InfoBox.tsx** — 5 variants per SPEC:
    - Props: `variant: 'blue' | 'hay' | 'green' | 'purple' | 'red'`, `title?: string`, `children`
    - blue: `bg-blue-lt` bg, blue title
    - hay: `bg-hay-lt` bg, `text-[#7A5C00]` title
    - green: `bg-[rgba(45,80,22,0.06)] border border-moss/20` bg, moss title
    - purple: `bg-purple-lt` bg, purple title
    - red: `bg-red-lt` bg, red title
    - Base: `rounded-[12px] p-4 border-[1.5px]`

    **ProgressBar.tsx** — gradient bar for wizards:
    - Props: `progress: number` (0-100), `className`
    - Container: `h-[3px] bg-fog rounded-full w-full`
    - Fill: `h-full rounded-full bg-gradient-to-r from-moss to-meadow transition-all duration-300` with width set to progress %

    - Props: `label?: string`, `checked?: boolean`, `onCheckedChange?: (checked: boolean) => void`, `disabled?: boolean`
    - Root: 34px wide, 18px tall pill shape (`w-[34px] h-[18px] rounded-full`)
    - Unchecked: `bg-fog`
    - Checked: `bg-moss`
    - Thumb: white circle, 14px, translates on checked (`translate-x-[16px]`)
    - Use Radix Switch.Root and Switch.Thumb

    - Props: `label?: string`, `checked?: boolean`, `onCheckedChange?: (checked: boolean) => void`, `disabled?: boolean`
