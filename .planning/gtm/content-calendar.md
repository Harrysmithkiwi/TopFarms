# Content calendar — productionised

*2026-07-08 · Stage-2 M3. Turns the 4-week theme cycle (skills-adoption.md) + the four
founder-brand pillars + the Seasonal Insights `sell_now` feed into a runnable weekly
production loop. Solo-founder budget: ~2.5 hrs/week total. Voice: config rules, always.*

## The weekly production loop (fits inside weekly-operating-rhythm.md)

| Slot | Output | Source | Time |
|---|---|---|---|
| Mon (with target-list block) | pick the week's theme: `sell_now` sector × pillar | Seasonal Insights + this table | 10 min |
| Tue | **Tuesday email** (one number · one story/stance · one ask) | Index/pipeline + pillar | 20 min |
| Tue + Thu | LinkedIn post ×2 = email recycled (pillar 1/4 cuts) | the email | 10 min |
| Wed + Fri | Owned-group post ×2 (seeker-flavoured, pillar 2/3) | rush-tier/retention drafts + variation kit | 20 min |
| Fri (review slot) | next-week theme + 1 carousel/asset ONLY if the queue is empty | content bank below | 30 min |

Kill rule mirrors founder-brand-plan: the email ships every Tuesday with true numbers,
however small. Everything else flexes in calving weeks; the email doesn't.

## Month themes (4-week cycle, seasonally steered)

Wk1 **We exist** (stance/why) · Wk2 **What we do** (product truth, workers-never-pay) ·
Wk3 **It works** (numbers/stories — real only; until placements exist, use study
patterns: "what 35 job-hunting posts told us") · Wk4 **Join us** (Founding-25 /
rush-tier CTA). The `sell_now` sector skins every theme (July–Sep = calving/relief
examples; Feb–May = hort/vintage once那 vertical is live).

## Content bank (make once, reuse)

- 1 carousel/month max (the existing why-good-workers-leave draft ships first —
  render it, don't redraft it). Next in queue per skills-adoption list, seasonally picked.
- Poster set + 2 videos already exist (over-supplied — produce NO new visual assets
  until distribution catches up; audit finding, respected here).
- Index monthly cut (when live) auto-becomes: email lead number + 1 LinkedIn post +
  1 group post + the public page. One artefact, four slots.

## Consent + truth gates (every piece)

Real names/farms → written consent on file (config rule). No invented metrics, no
future-dated claims, {{PLACEHOLDER}} tokens never ship. If a week has no true number
worth leading with, lead with a question to the list instead — asked honestly, replies
become next week's content.
