Failed runs auto-retry ×3. Slack alert on consecutive failures. Partial dataset saved to dataset even on crash. Run logs to Apify console.

Apify config note
Actor output format
Each actor outputs a standardised JSON envelope regardless of source:

{ source, scraped_at, raw_text, url, post_id, author }

Raw text is passed downstream untouched — no field extraction at this layer. That's Claude's job.

✓ Unique fingerprint ✓ Source identified ✓ Post type classified ⚠ Low text length (<80 chars) ⚠ Duplicate URL detected ✗ No contact info ✗ Non-NZ location ✗ Spam pattern match
Queue logic
Status-driven processing
A Supabase Edge Function polls for status = 'pending' rows every 5 minutes and dispatches batches to the AI parser.

Max batch size: 50 posts per invocation. Failed rows retry up to ×3 before being flagged for manual review.
Pending batch → Claude API · structured extraction prompt

Prompt engineering note
JSON-only output enforced
System prompt instructs Claude to respond only with valid JSON matching the schema — no preamble, no markdown fences. Response is parsed directly into the parsed_jobs table. Confidence scores (0–1.0) are included per field to flag low-confidence extractions for manual review.
Structured JSON → Supabase upsert

06 · Matching
Match scoring engine
Weighted multi-dimension scoring — produces the % match circles seekers see
Postgres function Triggered on update
🔄
Shed type match
Rotary ↔ Rotary = 100pts. Rotary ↔ HB = 40pts (transferable). AMS = separate pool. Highest single weight.
Weight: 25pts
Critical filter
📍
Location score
Same region = 100pts. Adjacent region = 60pts. Open to relocate = 80pts baseline. Inverse distance weighting on lat/lng.
Weight: 20pts
🏠
Accommodation match
Seeker needs housing + job offers housing = 100pts. Pet requirements, couples, family size all sub-scored separately.
Weight: 20pts
Critical filter
📜
Skills & quals
Required skills matched at full weight, preferred at 60%, bonus at 20%. DairyNZ level, tractor certs, herd size experience.
Weight: 20pts
💰
Salary alignment
Seeker min salary vs job offer. Partial credit for close ranges. Jobs above seeker minimum always score full.
Weight: 10pts
🛂
Visa / eligibility
NZ citizen/resident = full eligibility. Working holiday, student visa, employer sponsorship required — all tracked separately.
Weight: 5pts
👫
Couples / family
Couple seeking work + employer open to couples = bonus points. Partner's sector background also factored if provided.
Bonus pts
⚡
Recency boost
Jobs posted in last 7 days get a small freshness multiplier. Prevents stale listings dominating results.
Multiplier
Scored & ranked listings → TopFarms app layer
